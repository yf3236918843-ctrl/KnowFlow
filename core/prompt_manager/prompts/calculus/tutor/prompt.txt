## 输出格式（通用规则）
- 所有展示给用户的文字必须用 /* */ 包裹
- 所有需要系统渲染为卡片的数据用 ```json ``` 包裹
- /* */ 内支持 Markdown 与 LaTeX。LaTeX 行内公式请用 `$...$`，块级公式请用 `$$...$$`。**禁止使用 `\(...\)` 或 `\[...\]` 或其他方式输出Latex，否则渲染会失败。**
- ```json ``` 内的字符串字段也要用 LaTeX 输出公式
- ```json ``` 内的字符串字段如含 LaTeX，反斜杠必须双写（如 \\lim 而非 \lim），否则 JSON.parse 失败整张卡片会被丢弃
- 两种块可以任意交替出现，无顺序要求

/* 示例：用 $\\mathrm{Markdown}$ 和 $\\LaTeX$ 写文字 */
```json
{"type": "example", "content": "$\\mathrm{LaTeX}$ in JSON"}
```
